# LJT-Homepage

Personal academic homepage built with Jekyll and the Academic Pages template.

## About

This is the personal website of **Junteng Liu**, a first-year PhD candidate at the HKUST NLP Group.

## Local Development

To run this site locally:

1. Install dependencies: `bundle install`
2. Serve the site: `bundle exec jekyll serve`
3. Open your browser to `http://localhost:4000`
